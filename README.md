# Bigdata modeling
Data Engineering Project: Big Data modeling using postgreSQL


## prerequisite

* PostgreSQL
* Python 3.6 or higher


## Dependencies
```
pip install -r requirements.txt
```

## DB Setup
```
export DB_USER = <PostgreSQL DB user>
export DB_USER = <PostgreSQL DB user>
python create_tables.py 

```

## Run Data pipeline

python etl.py
